- **Multi-Tenant School Onboarding & AI Migration:** A centralized dashboard for managing multiple school entities, featuring an AI-powered data importer that automatically maps and cleans messy legacy student/staff records from spreadsheets for instant setup.
- **Zero-Fee "Sovereign" Payment System:** A financial module that eliminates transaction fees by generating locked UPI payment links for parents and provides a reconciliation tool for administrators to verify payments by matching bank statements with student-submitted transaction IDs.
- **Hybrid Attendance & Geofencing:** A location-aware tracking system that uses GPS boundaries to verify staff presence and an offline-first mobile interface for teachers to quickly mark student attendance, syncing data automatically once connectivity is restored.
- **Automated Academic Engine & Digital Report Cards:** A flexible grading tool where teachers input raw scores and the system automatically calculates weighted averages, assigns grades based on school-specific rules, and generates professional PDF report cards for instant publishing to parents.
- **Unified Communication & Virtual Classroom:** A cost-effective engagement suite that uses Telegram bots for free automated alerts (fees, attendance, announcements) and includes an integrated video conferencing tool for hosting secure online classes directly within the platform.
